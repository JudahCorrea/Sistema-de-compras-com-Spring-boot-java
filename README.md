# Sistema de Compras
Um sistema de compras baseado em um diagrama de casos de uso e com comunicação com banco de dados  PostgreSQL.

# Conhecimentos Utilizados 
* End-points
* Docker conteiner
* Conection class
* three-tier architecture

# Conhecimentos Almejados
* uniform interface 
* stateless
* cacheable
* client-server
* layered system
